## Was ist Bazzy?

Bazzy ist eine lokale Kassensoftware für die Durchführung von Kinder- und Schulbasaren. Die App verbindet sich mit einem separaten Bazar-Server, der die Daten lokal verwaltet. Alle Daten bleiben auf dem Gerät des Veranstalters – es werden keine Daten an den Entwickler übermittelt.

**Funktionsumfang (Auszug):**
- Mehrere Kassen gleichzeitig (Netzwerkbetrieb)
- Teilnehmerverwaltung mit Nummernzuweisung
- Automatische Abrechnung und Teilnehmerrechnungen als PDF
- Bon-Druck und Bon-Historie
- Auswertungen und Exportfunktionen
- Annahme- und Abholungslisten
- Einnahmen-/Ausgabenverwaltung
- Live-Übersicht für Veranstalter
- Automatische Updates
